Stap 0: Development Setup

Voordat we beginnen, moeten we onze development omgeving klaarzetten:

1. Visual Studio Code installeren

Download en installeer Visual Studio Code (VSCode) als je dit nog niet hebt:

1. Ga naar https://code.visualstudio.com/
2. Download de juiste versie voor jouw besturingssysteem
3. Voer het installatieprogramma uit en volg de instructies

2. Live Server extensie installeren

De Live Server extensie zorgt ervoor dat we onze website direct kunnen bekijken tijdens het ontwikkelen:

1. Open VSCode
2. Klik op het extensie-icoon in de linker balk (of druk op Ctrl+Shift+X)
3. Zoek naar "Live Server"
4. Klik op "Installeren" bij de extensie van Ritwick Dey

3. Projectmap openen

Maak een nieuwe map voor je project en open deze in VSCode:

1. Maak een nieuwe map aan (bijvoorbeeld "quiz-project")
2. Open VSCode
3. Klik op "File" > "Open Folder" en selecteer je nieuwe map
4. Maak de volgende mappen aan in je project:
   - css/
   - js/
   - js/data/
   - assets/images/questions/
     - games/
     - flags/
     - sport/

4. Live Server starten

Als je klaar bent met het maken van je HTML-bestand:

1. Open je HTML-bestand in VSCode
2. Klik rechtsonder in VSCode op "Go Live"
   - Of: klik rechts op het bestand in de explorer en kies "Open with Live Server"
3. Je browser opent automatisch met je website
4. De website herlaadt automatisch wanneer je wijzigingen opslaat

Nu zijn we klaar om te beginnen met het bouwen van onze quiz!

Stap 1: Basis HTML Structuur

Laten we beginnen met het maken van de basisstructuur van onze quiz. Volg de onderstaande stappen:

1. Maak een nieuw HTML-bestand aan

Open je projectmap in Visual Studio Code. Klik met de rechtermuisknop op de map en kies New File. Geef het bestand de naam game.html.

2. Voeg de standaard HTML-structuur toe

Open het bestand game.html. Typ een uitroepteken (!) en druk op de Tab-toets. Dit genereert automatisch de basis HTML-structuur:

Stap 3: Quiz Data

Maak een data.js bestand aan in de js/data map en kopieer de volgende code. Hieronder wordt uitgelegd wat elk deel van de data doet:

1. Structuur van de data

De data is een JavaScript-object dat alle informatie bevat voor de quiz. Het is opgebouwd uit een lijst van thema's, elk met een unieke id, een name, en een lijst met questions.

2. Thema's

Elk thema bevat:

  • id: Een unieke identifier voor het thema (bijvoorbeeld "games").
  • name: De naam van het thema (bijvoorbeeld "Game-covers").
  • questions: Een lijst met vragen die bij dit thema horen.

3. Vragen

Elke vraag bevat:

  • id: Een unieke identifier voor de vraag (bijvoorbeeld "q-1-1").
  • question: De tekst van de vraag.
  • image: De URL naar een afbeelding die bij de vraag hoort.
  • answers: Een lijst met mogelijke antwoorden.

4. Antwoorden

Elk antwoord is een object met:

  • text: De tekst van het antwoord.
  • isTrue: Een boolean die aangeeft of het antwoord correct is (true) of niet (false).

function setupQuiz(button, themeId) {
  button.addEventListener("click", () => {
    theme = themeId;
    $quizIntro.classList.add("hidden");
    $gameIntro.classList.remove("hidden");

    const themeData = data.themes.find(
      (themeData) => themeData.id === themeId
    );
    if (!themeData) return;

    let currentQuestionIndex = 0;

    function showQuestion() {
      const current = themeData.questions[currentQuestionIndex];

      $gameIntro.innerHTML = `
        
${current.question}

        
        

          ${current.answers
            .sort(() => 0.5 - Math.random())
            .map((answer) => ``)
            .join("")}
        

      `;

    }

    showQuestion();
  });
}

Voeg tot slot deze code toe onderaan het bestand om de quizknoppen te activeren:

setupQuiz($gamesButton, "games");

Stap 8: Volledig JavaScript Bestand

Hieronder vind je het volledige JavaScript-bestand dat alle stappen combineert. Kopieer deze code naar je main.js bestand in de js map:

import { data } from "./data/data.js";

const $gamesButton = document.getElementById("games");
const $sportsButton = document.getElementById("sports");
const $countryButton = document.getElementById("countries");

const $quizIntro = document.getElementById("intro");
const $gameIntro = document.getElementById("game-intro");

if (
  $gamesButton &&
  $sportsButton &&
  $countryButton &&
  $quizIntro &&
  $gameIntro
) {
  let theme = "";
  let rightAnswers = 0;
  let questions = data;

  function setupQuiz(button, themeId) {
    button.addEventListener("click", () => {
      theme = themeId;
      $quizIntro.classList.add("hidden");
      $gameIntro.classList.remove("hidden");

      const themeData = data.themes.find(
        (themeData) => themeData.id === themeId
      );
      if (!themeData) return;

      let currentQuestionIndex = 0;

      function showQuestion() {
        const current = themeData.questions[currentQuestionIndex];

        $gameIntro.innerHTML = `
          
${current.question}

          
          

            ${current.answers
              .sort(() => 0.5 - Math.random())
              .map((answer) => ``)
              .join("")}
          

        `;

        document.querySelectorAll(".answer").forEach((button) => {
          button.addEventListener("click", () => {
            document
              .querySelectorAll(".answer")
              .forEach((btn) => (btn.disabled = true));
            if (
              button.textContent === current.answers.find((a) => a.isTrue).text
            ) {
              button.style.backgroundColor = "green";
              rightAnswers++;
            } else {
              button.style.backgroundColor = "red";
            }

            currentQuestionIndex++;
            if (currentQuestionIndex < themeData.questions.length) {
              setTimeout(showQuestion, 1500);
            } else {
              setTimeout(() => {
                $gameIntro.innerHTML = `
                  
Quiz afgelopen!

                  
Je hebt ${rightAnswers} van de ${themeData.questions.length} vragen goed beantwoord.

                `;
              }, 1500);
            }
          });
        });
      }

      showQuestion();
    });
  }

  setupQuiz($gamesButton, "games");
  setupQuiz($sportsButton, "sports");
  setupQuiz($countryButton, "flags");
}

Met deze code is je quiz volledig functioneel. Zorg ervoor dat je de juiste HTML-structuur en CSS-styling hebt toegevoegd om alles correct te laten werken.
