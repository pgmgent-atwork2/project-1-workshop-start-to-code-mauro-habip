2. De HTML-structuur Maken

HTML is het skelet van je webpagina. Kopieer deze code naar je index.html:


<!DOCTYPE html>
<html lang="nl">
<head>
    <meta charset="UTF-8">
    <title>Quiz Spel</title>
    <link rel="stylesheet" href="./css/main.css">
</head>
<body>
    <header class="header">
        <h1>De Slimste Student</h1>
    </header>

    <main>
        <div class="game-intro hidden" id="game-intro"></div>
        <section class="intro" id="intro">
            <button class="btn btn-blue" id="sports">Sport</button>
            <button class="btn btn-yellow" id="countries">Vlaggen</button>
            <button class="btn btn-orange" id="games">Games</button>
        </section>
    </main>

    <footer class="footer">
        <p>Gemaakt door Jou!</p>
    </footer>
    <script src="./js/main.js" type="module"></script>
</body>
</html>

Dit maakt:

  • Een header met de titel van het spel
  • Drie categorieknoppen
  • Een ruimte waar vragen verschijnen
  • Een eenvoudige footer

3. Stijlen Toevoegen met CSS

CSS zorgt ervoor dat je spel er mooi uitziet. Voeg dit toe aan css/main.css:


body {
  background-color: #fdc674;
  font-family: Arial, sans-serif;
}

.intro {
  display: flex;
  justify-content: center;
  margin: 2rem auto;
  width: 80%;
  text-align: center;
}

.game-intro {
  width: 80%;
  margin: 2rem auto;
  background: white;
  padding: 2rem;
  border-radius: 10px;
}

.btn {
  border: none;
  color: white;
  padding: 1rem 2rem;
  margin: 0.5rem;
  font-size: 1.2rem;
  cursor: pointer;
  border-radius: 5px;
}

.btn-blue { background-color: #008cba; }
.btn-orange { background-color: #ff9800; }
.btn-yellow { background-color: rgb(230, 230, 20); }

.hidden { display: none; }

.header, .footer {
  text-align: center;
  padding: 1rem;
}

Dit zal:

  • Je pagina een mooie oranje achtergrond geven
  • Je knoppen stijlen met verschillende kleuren
  • Je inhoud centreren
  • Het quizgebied er netjes uit laten zien
